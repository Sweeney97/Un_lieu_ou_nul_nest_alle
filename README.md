# Page de présentation

Bienvenue sur le dépôt GitHub de ma création pour l'évaluation de mi-session; j'ai opté pour l'écriture codée.

J'ai écrit une histoire à l'aide d'Inform 6, intitulée « Un lieu où nul n'est allé ».

Si vous ne savez pas ce qu'est Inform, je vous invite à consulter [ce fichier](https://github.com/Sweeney97/EDN6003/blob/main/premiercontact.md) avant de lancer le jeu, qui indique quelques manipulations de base.

Il y a aussi [la transcription du jeu](https://github.com/Sweeney97/EDN6003/blob/main/transcript.md), qui sert autant à lire la séquence du jeu (si vous n'avez pas envie d'y jouer) que de guide pour vous débloquer. **Gardez-la donc à portée**.

Vous pouvez [jouer directement en ligne](https://iplayif.com/?story=https://github.com/Sweeney97/EDN6003/raw/main/EDN6003-V1.1.z5) (grâce à [Parchment](http://iplayif.com/)). Si vous préférez pouvoir interrompre votre partie et y revenir, vous devrez [télécharger le jeu](https://github.com/Sweeney97/EDN6003/raw/main/EDN6003-v1.0.z5) et un logiciel pour exécuter le fichier, je vous conseille [Lectrote](https://www.electronjs.org/apps/lectrote). Sachez que le jeu dure une quinzaine de minutes environ.

Finalement, vous avez aussi accès [au code source](https://github.com/Sweeney97/EDN6003/blob/main/EDN.inf).
Si Inform vous intrigue, j'ai créé [ce fichier d'information](https://github.com/Sweeney97/EDN6003/blob/main/presentationinform.md).
