Bienvenu sur le dépôt GitHub de ma création pour l'évaluation de mi-session du cours FRA3715, *Écriture et nouveaux médias*; j'ai opté pour l'écriture codée.

J'ai écrit une histoire à l'aide d'Inform 6, intitulée *Un lieu où nul n'est allé*.

Si vous ne savez pas ce qu'est Inform, je vous invite à consulter [ce fichier](https://github.com/Sweeney97/Un_lieu_ou_nul_nest_alle/blob/master/premier_contact.md) avant de lancer le jeu, qui indique quelques manipulations de base.

Il y a aussi [la transcription du jeu](https://github.com/Sweeney97/Un_lieu_ou_nul_nest_alle/blob/master/transcript.md), qui sert autant à lire la séquence du jeu (si vous n'avez pas envie d'y jouer) que de guide pour vous débloquer. **Gardez-la donc à portée**.

Vous pouvez [jouer directement en ligne](https://iplayif.com/?story=https://github.com/Sweeney97/Un_lieu_ou_nul_nest_alle/raw/master/EDN6003-v.1.3.z5) (grâce à [Parchment](http://iplayif.com/)). Si vous préférez pouvoir interrompre votre partie et y revenir, vous devrez [télécharger le jeu](https://github.com/Sweeney97/Un_lieu_ou_nul_nest_alle/raw/master/EDN6003-v.1.3.z5) et un logiciel pour exécuter le fichier, je vous conseille [Lectrote](https://www.electronjs.org/apps/lectrote). Sachez que le jeu dure une quinzaine de minutes environ.

Finalement, vous avez aussi accès [au code source](https://github.com/Sweeney97/Un_lieu_ou_nul_nest_alle/blob/master/EDN6003.inf).
