# Page de présentation

Bienvenue sur la page de ma création pour l'évaluation de mi-session; j'ai opté pour l'écriture codée.

J'ai écrit une histoire à l'aide d'Inform 6.

Si vous ne savez pas ce qu'est Inform, je vous invite à consulter cette page avant de lancer le jeu, qui indique quelques manipulations de base.

Il y a aussi la transcription du jeu, qui sert autant à lire la séquence du jeu (si vous n'avez pas envie d'y jouer) que de guide pour vous débloquer. Gardez-la donc à portée.

Vous pouvez jouer directement en ligne, grâce à [Parchment](http://iplayif.com/). Si vous préférez pouvoir interrompre votre partie et y revenir, vous devrez télécharger le jeu et un logiciel pour exécuter le fichier, je vous conseille Lectron.

Finalement, vous avez aussi accès au code source ici.
Si Inform vous intrigue, j'ai créé cette petite page d'information.
